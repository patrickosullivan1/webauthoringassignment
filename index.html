Introduction to HTML by Patrick O'Sullivan

Doctype

The declaration is the first line of the code in HTML or XHTML document. It specifies the HTML version used in the document.

 <!doctype html>

Example 

 <!DOCTYPE html>

 <html>

 <head> 

<title>Rugby Stats</title> 

</head> 

<body> <p>Welcome To My Stats Page</p>

 </body>

 </html>

Output

Welcome To My Stats Page  

html

HTML (HyperText Markup Language) is a primary markup language for creating websites. It consists of a series of codes used to structure texts, images, and other content to be displayed in the browser. It is the first and last tag on the page after the Doctype

 <html></html>

head

The head element is a container for metadata (data about data) and is placed between the html tag and the body tag.

Metadata typically define the document title, language, keywords (for search engines) and is not displayed on the webpage

<head></head> 

body

The body is the visible part of the webpage. All text and images included between the body/close body tags will be displayed on the webpage. In the above example the visible part of the page would be 

 Welcome To My Stats Page 

<body></body> 

title

The title tag defines the title of the document and must be text only. It is shown in the browser page tab. In the above example Rugby Stats would be the tab title  

<title></title> 

p

<p>This is a paragraph</p>

A paragraph generally is a block of text in the webpage. It may contain images and links within the block.      

This is a paragraph within this document

span

The span is an inline  container used to style part of a sentence or paragraph

  <p>This is my <span style ="color:red">first</span> HTML document</p>

 In this case the word first will be red.

This is my first HTML document

 

Div

<div></div> 

The div tag divides a section within the html document. 

 The document can contain as many div ids or blocks as needed. Different ids will need to be assigned to each block within the document and each can then be styled differently e.g 

<div id="section1 "></div>  when styling this div in css it must be preceded by the # symbol    #section1{}. The style will be applied to the elements within the div

<div id="section2"></div> when styling this div in css use      #section2 {}

A div class can be used to style more than one section within a document. This can be used to style different elements that may appear in many of the different blocks e.g you may want to style all quotations in the document in the same way.

You would then assign a class to quotations e.g.

<div class="quotations"> </div>

When styling this div in css or inline it must be preceeded by a full stop    .quotations{}  

The class can be used throughout the document whereas the id represents its own block.   

All the text within this div is styled to have a background color of aquamarine

The div is setup with that style

This is the last sentence within the div

Formatting elements - strong, emphasis

<strong></strong> The strong tag is used to highlight a word or a sentence. The formatted text appears as bold.

<p>This is my <strong>first</strong> HTML document</p> 

This is my first HTML document

<em></em> The em or emphasis tag is used to showemphasis on a word or sentence. The text appears italised

images and alt

the <img> tag is used to embed images in web pages. Images are linked and the webpage must be told where to go find the. Generally this is in an images subfolder within the website folder. An images folder has been created for this purpose. Height and width can be specified in the code also.  the <img tag closes itself/>

alt specifies text that will be instead of the picture in case the picture cannot be show. In this case the alt is baseball brawl 

 <img src="images/baseballbrawl.jpg" width="200" height="200" alt="baseball brawl"/>

 

links

In the header with the other meta data a style sheet .css file can be linked to the html document. This style sheet will be used instead of inline styling to style the elements of the webpage.

 <link href="styles.css" rel="stylesheet" type="text/css">

href specifies the location of the file - generally in the site folder. rel  specifies the relationship i.e. stylesheet for the html file and type is the media type i.e. what is the output device.

This link is invisible on the webpage itself and cannot be opened from the browser.

Links to other web pages either within or without the site are created use the 

<a> anchor  ag 

The tag must also have a href to specify the page that you want to link to

 <p><a href="#" target="blank">cssassignment</a></p>

cssassignment

Clicking on the link just above will take you to cssassignment.html - the second page created for the website -  and target blank means that it will open in a new tab and this page will remain open also - so two pages open at once. If you wanted the linked page to open instead of this page the target would be "self".               

quote and blockquote

To put text in quotation marks the tag <q></q> is used within the paragraph 

 <p> Richard Nixon famously said <q>I am not a crook</q> when leaving the White House in disgrace </p>

Richard Nixon famously said I am not a crook when leaving the White House in disgrace

Blockquote <blockquote></blockquote> is generally used for longer quotes  and appears indented on the page  

 <blockquote>  Have respect for yourself Have respect for the officials Have respect for your opponents Have respect for the game and written/unwritten rules </blockquote> 

Any tabs, line spaces or line breaks will not appear   

Have respect for yourself Have respect for the officials Have respect for your opponents Have respect for the game and written/unwritten rules

abbreviations

The <abbr></abbr> tag defines an abbreviation or an acronym e.g. FAI

 <p><abbr title="Football Association of Ireland">FAI</abbr></p>

Hover over the abbreviation and the full title will be visible 

FAI

meta

The <meta> tag is displayed in the <head> section 

<meta> displays metadata - data about other data e.g

 <meta charset="utf-8"> is the character ecoding, storage and transfer protocol. 

It also contains the keywords which search engines use to rank websites in searches. This is important when creating your site as you want to be viewed as much as possible.

 <meta name="keywords" content="latest news, breaking news, Ireland news"> may be keywords for a news website that wants viewers to visit for the most up to date news.

Other data that can be included here is the name of the author or the frequency that the site refreshes.

This data does not appear on the site.  

forms

The <form></form> tags are used to solicit information from the user. Any information can be requested - name, address, phone number, email. Preferences can be sought either one or more.

The <form> element is a container for inputting data such as name, address, phone number, selecting preferences and options.

Between the <form></form> tags  the<fieldset></fieldset> tags are set up.Within this <label></label> tags are written. Before  the label is applied the <input type =""> is written and closed. The label or name of the field is entered.

Input type can be 'text' for names and addresses, can be 'email', 'password' (hidden) or 'tel' for telephone number.  

Input can be 'checkbox' where multiple options can be selected.

Input is 'radio' where one option has to be chosen from more than one.
